hugoでshortcodeを用いてtableにcssを適応

久しぶりにHugoの投稿をしようと思います。 今回の内容は、

「マークダウン形式で書いたtableに対して、cssを適応させる」 です。

例えば、マークダウン形式でテーブルを書くと、以下のようになります。 

| 選手 | 背番号 | 国 |
|----------|----------|----------|
| 古橋 亨梧 | 16 | 日本 |
| アンドレス・イニエスタ | 8 | スペイン|
| トーマス・フェルマーレン | 4 | ベルギー |

↓これをそのままhtmlに変換したのが以下になります↓

選手 背番号 国
古橋 亨梧 16 日本
アンドレス・イニエスタ 8 スペイン
トーマス・フェルマーレン 4 ベルギー

\

↑ここまで↑

うぅん見にくい。

こいつに今回紹介する方法でテキトーなcssを適応すると以下のようになります。

マークダウンで書いたテーブルにcssを適応した結果

選手 背番号 国
古橋 亨梧 16 日本
アンドレス・イニエスタ 8 スペイン
トーマス・フェルマーレン 4 ベルギー

.mdファイルに直接テーブルタグを書いてclass適応してっていうのがめんどくさいので、 できる限りマークダウン形式でテーブルを書いて shortcode を用いて、classを適応をします。 結論から書いていきます。

Markdownの中身（記事のもとになる.mdファイル）

{{< mdtable class = "simple-table" >}}
| 選手 | 背番号 | 国 |
|----------|----------|----------|
| 古橋 亨梧 | 16 | 日本 |
| アンドレス・イニエスタ | 8 | スペイン|
| トーマス・フェルマーレン | 4 | ベルギー |
{{< mdtable >}}

Shortcodeの中身（mdtable.html）

{{if (.Get "class")}}
    <div class = "{{ .Get "class" }}">
{{ else }}
    <div class = "simple-table">
{{ end }}
{{.Inner | markdownify }}
</div>

適応したcss

div.simple-table table {
	width: 100%;
	border-top: 1px solid #e2e2e2;
	border-left: 1px solid #e2e2e2;
	background: #ffffff;
	margin-bottom:15px;
  	margin-left: auto;
  	margin-right: auto;
     color:#000000;
}

div.simple-table table th{
	border-right: 1px solid #e2e2e2;
	border-bottom: 1px solid #e2e2e2;
 	padding:10px 0;
  text-align: center;
}
div.simple-table table td {
	font-size: 0.9em;
	border-right: 1px solid #e2e2e2;
	border-bottom: 1px solid #e2e2e2;
  text-align: center;
  padding:10px 0;
}
div.simple-table table th {
	background-color: #f3f0f0;
}

ざっくりとした説明

最初は{{replace}} を使って{{ {.Inner | markdownify }}で、 マークダウン形式からマークアップに変換し、受け取った<table>の中身を 書き換える方法を考えていました。しかしあまりうまく行きません。 ですので回避策として、shortcode側で<table>の外側にクラスを適応した <div>をかまして、 cssで対応させる手法をとりました。 処理の流れ等の詳しい説明は今回は割愛いたします～。
