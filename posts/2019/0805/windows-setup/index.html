【超個人的】Windows初期インストールソフト

最近、合同研究やインターンなどに行かせて頂く機会が増え、 PCのセットアップをする機会も増えました。 そしてその都度、「あれ？何入れてたっけ？」ってなるのでまとめようと思います。

主なインストールソフト一覧

  • Chrome：ブラウザ
  • chocolatey：Package管理
  • atom：エディタ
  • Mery：エディタ
  • anaconda3
  • MeCab：形態素解析
  • ElasticStack
  • Cica：フォント
  • 7-zip

Chrome

Microsoft Edgeの最大の利用目的と言っても過言ではない Chrome のインストール。 インストールはこちらからhttps://www.google.com/intl/ja/chrome/ インストール完了後、 設定 → アプリ → 規定のアプリ → Webブラウザー から Chromeを選択する。この時、Microsoftからの「ちょっとまったぁ！」が 入るけども気にせず変更を完了する。

chocolatey

まず管理者用コマンドプロンプトをひらく。

  • Windowsキー(Windowsのマークが書かれたボタン)とrの同時押し
  • 表示された入力欄※1に cmd を入力し ctrl shift Enter を同時押し
  • 「このアプリがデバイスに変更を加えることを許可しますか」というメッセージが出てくるが気にせずOKを押す

以下のコマンドをコピーしてコマンドプロンプトに ペーストすればインストールが始まります。 始まらなければエンターを推してください。

@"%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe" -NoProfile -InputFormat None -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"

表示が終わればインストール完了です。

続いて念のために以下のコマンドでchocolateyをアップデートしておいてください。

choco upgrade chocolatey

atom

jupyterを使う時以外に、ゴリゴリ何かの作成をするときはatomを使っています。

メインインストール

chocoを利用することで簡単に入れることができます。

choco install atom

atomの設定

コア設定

  • ビープ音を鳴らす：オフ
  • 自動アップデート：オフ
  • 削除されたファイルのタブを閉じる：オン
  • 空になったウィンドウとタブを同様に閉じる：オフ
  • 起動時に前回のウィンドウ表示状態を復元する：オフ

エディタ設定

  • フォント：Cicaに設定
  • ソフトタブ：オン
  • タブ幅：4（jupyter）

システム設定

システム設定はすべてオンにする

Mery

めちゃくちゃ軽いテキストエディタ。

choco install mery

anaconda3

インストール

データを扱う研究をするので、鉄板のAnaconda3でpythonの環境を作る。

choco install anaconda3

jupyterの拡張機能

素晴らしいまとめ記事

pip install jupyter_contrib_nbextensions
  • table of contents
  • isort formatter
  • Move selected cells

MeCab

よく使う形態素解析器。 最近はSudachiも魅力。

pythonで利用するためにWindows64bit版のものを入れる。 https://github.com/ikegami-yukino/mecab/releases/tag/v0.996.2

pythonで使えるようにする

pip install mecab-python-windows

ElasticStack

個人的によく使う全文検索エンジンと関連ツール。 詳細はまた今度。

各種インストールurl

elastichsearch https://www.elastic.co/jp/downloads/elastichsearch kibana https://www.elastic.co/jp/downloads/kibana logstash https://www.elastic.co/jp/downloads/logstash

Cica

プログラミング用のフォント。 個人的に見やすくて好きです。

https://github.com/miiton/Cica/releasesここからダウンロードする。 解凍したファイルの中に.ttfがあるので C:\Windows:Fontsにドラッグアンドドロップ。

7-zip

圧縮ファイルの展開やファイルの圧縮ができる便利ツール。

choco install 7-zip
